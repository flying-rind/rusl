# isascii 函数规约

## 复杂度分级: Level 1

---

## 函数接口

```rust
use core::ffi::c_int;

extern "C" fn isascii(c: c_int) -> c_int;
```

### 前置/后置条件

**[Pre-condition]:**
`c`: 类型为 `c_int`，任意整数值。无其他前置条件。

**[Post-condition]:**
- Case 1: `c` 在 0 到 127 范围内（bit 7 及以上全为零，即 7 位 ASCII 字符）
  - 返回非零值（通常为 1）。
- Case 2: `c` 超出 0-127 范围（bit 7 或更高位被设置），或 `c` 为负数
  - 返回 0。

### 不变量

**[Invariant]:** 纯函数。无内部状态，无 locale 依赖。

### 意图

判断字符是否为 7 位 ASCII 字符。这是 BSD/POSIX 兼容函数。现代代码中通常不需要此函数（直接检查 `c <= 0x7f` 即可），但为兼容性需要提供。

在 Rust 内部实现中，可使用等价位运算 `(c & !0x7f) == 0` 或 `c as u32 <= 0x7f`。

### 系统算法

```
计算 c & ~0x7f，若结果为零则返回非零值（真），否则返回 0（假）。
即检查 c 的高位（bit 7 及以上）是否全为零。
时间复杂度 O(1)。
```
